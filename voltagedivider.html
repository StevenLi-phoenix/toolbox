Formulas

Unloaded: Vout = Vin · R2 / (R1 + R2)
Quiescent current: I = Vin / (R1 + R2)
Power: PR1 = I² · R1,   PR2 = I² · R2
Loaded: Vout = Vin · (R2‖RL) / (R1 + R2‖RL),   R2‖RL = R2·RL/(R2+RL)
Thévenin: Rth = R1 ‖ R2,   Vth = Vout (unloaded)
