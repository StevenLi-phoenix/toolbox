ℹ️ About SSE

Server-Sent Events (SSE) provide a simple way to push real-time updates from server to client over HTTP. Unlike WebSockets, SSE is one-directional (server → client), uses standard HTTP, supports automatic reconnection, and works through proxies. The browser's EventSource API handles connection management. Common use cases include live feeds, AI streaming responses, notifications, and log tailing.
