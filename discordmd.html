Syntax Cheat Sheet

  • **bold** · *italic* · __underline__ · ~~strike~~
  • # H1 · ## H2 · ### H3 (must start the line)
  • > quote · >>> block quote
  • `inline code` · triple backticks for code blocks with optional language
  • ||spoiler|| — click to reveal
  • Mentions: <@123> user, <@&123> role, <#123> channel
  • Timestamps: <t:1700000000:R> (relative, short, long)
  • Custom emoji: <:name:123> · <a:name:123> (animated)
