    About magnet URIs

    • xt (exact topic) is required — usually urn:btih:<hash> for BitTorrent v1 or urn:btmh:1220<hash> for BitTorrent v2 (multihash, SHA-256).
    • dn is the display name. xl is the exact length in bytes.
    • tr lists trackers, ws lists HTTP/FTP web seeds (BEP-19), as lists acceptable source URLs.
    • x.pe lists known peers, so (select-only) selects files in a multi-file torrent.
    • Multiple values of the same key are allowed. All values are percent-encoded.
