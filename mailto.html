RFC 6068 rules

    

      Multiple recipients are comma-separated. The first recipient goes in the path; headers (cc,
      bcc, subject, body) go in the query string and must be %xx-encoded.
      Newlines in the body become %0D%0A. Not every mail client honors bcc — Gmail web ignores it.
      Mobile mail apps usually handle the link well; desktop clients depend on OS default.
