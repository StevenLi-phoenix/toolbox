About

  

    Utility types are predefined generic type aliases in lib.es5.d.ts (and friends) that derive a new type from an existing one
    using mapped, conditional, indexed-access, and template-literal features of the type system. The cheat sheet on the left covers all 21
    types that ship with TypeScript 5.x. Hover an example in your IDE to see the canonical resolved form — the expander here covers
    the structural cases (object shape transforms, union filtering, string-literal manipulation) so you can sanity-check the resulting shape
    without leaving the browser.
