How it works
  • Add when a key or array slot exists only in the updated JSON.
  • Remove when a key or array slot exists only in the original JSON.
  • Replace when the value type or value changes.
  • Objects are compared recursively. Arrays are compared by position to keep output stable and easy to reason about.
