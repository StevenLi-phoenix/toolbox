About JSON Pointer (RFC 6901)

A JSON Pointer is a slash-delimited path into a JSON document. The empty pointer "" refers to the root. Each reference token is separated by /. Array indices are plain integers. Special characters must be escaped: ~ becomes ~0 and / becomes ~1. A pointer that ends with /- refers to the element after the last in an array (used by JSON Patch).
