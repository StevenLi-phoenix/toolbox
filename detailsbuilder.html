About <details> & the name= attribute

  
    
    
  • <details> + <summary> is the only built-in disclosure widget — zero JS, keyboard accessible, screen-reader friendly.
    • 
    
  • name="faq" Chrome 120+, Safari 17.2+, Firefox 130+ turns a group into an exclusive accordion: opening one closes the others. No JS required.
    • 
    
  • Animate height with interpolate-size: allow-keywords Chrome 129+ — transitions to height: auto now Just Work.
    • 
    
  • Style the marker with summary::marker or ::-webkit-details-marker { display:none } (Safari needs both).
    • 
    
  • For semantic FAQ pages, wrap items with FAQPage JSON-LD schema so search engines render rich results.
