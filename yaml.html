How to Use

  • Paste YAML or JSON in the left panel
  • Click YAML → JSON or JSON → YAML
  • Use ⇄ to swap input/output and keep editing
  • Supports multi-document YAML (--- separators)
  • All processing happens in your browser — nothing is sent to a server

Features

  • Instant syntax validation with error messages
  • Configurable JSON indent (2/4/tab/minified)
  • YAML flow level control (block vs inline)
  • Multi-document YAML support
  • One-click copy and download
