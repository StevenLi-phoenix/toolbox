💡 Common Patterns

  • Graceful stop → SIGTERM (15). Wait a few seconds, then SIGKILL (9) if still alive.
  • Reload config → SIGHUP (1). Common for nginx, haproxy, sshd.
  • Ctrl+C sends SIGINT (2) to the foreground process group.
  • Ctrl+Z sends SIGTSTP (20) — job-control stop. Resume with fg.
  • Core dump → SIGQUIT (3) or SIGABRT (6).
  • Check if alive → kill -0 <pid> (signal 0 tests delivery without sending anything).
