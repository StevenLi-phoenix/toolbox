About this Shuffler

Shuffles use the Fisher–Yates algorithm with a seeded Mulberry32 PRNG. The seed is hashed via cyrb53 so any string produces a uniform 32-bit state. Identical seeds yield identical shuffles — useful for testing card-game logic, reproducing bug reports, or sharing a specific deal with someone else. For real-money or cryptographic use, do not use this; it is deterministic, not secure.
