About Color Temperature

Color temperature describes the color of light emitted by an idealized black body at a given temperature in Kelvin. Lower values (1000–3000 K) appear warm orange-red, mid values (~5000–6500 K) are neutral daylight, and higher values (>8000 K) shift cool blue. This tool uses Tanner Helland's approximation, which matches CIE 1960 chromaticity closely in the 1000–40000 K range.
