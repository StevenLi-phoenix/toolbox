Glob Cheatsheet

*
Match any chars except /
**
Match any dirs recursively
?
Match exactly one char
{a,b}
Match a or b (brace expansion)
[abc]
Match one char in set
[!abc]
Match one char NOT in set
!(pat)
Match anything except pat
!pattern
Negate entire pattern
