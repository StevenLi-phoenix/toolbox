CODEOWNERS syntax cheat sheet

  

    Last matching rule wins. GitHub walks the file top-to-bottom and the last non-overridden pattern that matches the changed file decides the owners. Earlier rules are silently shadowed when a later one matches the same path.
  

  
    
    
  • * — anything in this directory and below
    • 
    
  • *.js — any .js file at any depth
    • 
    
  • /build/ — only the top-level build/ directory
    • 
    
  • docs/* — files directly inside docs/ (no recursion)
    • 
    
  • apps/**/*.tsx — any TSX file under apps/ at any depth
    • 
    
  • Owners are @username, @org/team, or user@example.com
    • 
    
  • Lines starting with # are comments
    • 
    
  • [Section Name] = required reviewers; ^[Optional Section] = optional
    •
