INI File Parser

Parse, edit, and convert INI configuration files ([section] / key = value) to and from JSON. Supports comments (; and #), quoted values, nested sections via dot notation, and array-style repeated keys. Runs entirely in your browser.

Notes

  • Lines starting with ; or # are comments and are stripped on parse.
  • Values outside quotes are trimmed. Use "…" or '…' to preserve whitespace and leading #/;.
  • Keys before any section are placed at the root of the JSON object.
  • Dot-notation sections like [a.b.c] are expanded to nested objects when the option is on.
  • No network requests — all parsing happens locally.
